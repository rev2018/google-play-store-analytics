About This Project

This dashboard visualizes analytics from the Google Play Store dataset. We explore insights like ratings vs reviews, content ratings, genre distribution, download trends and more. All visualizations were built in Python (Colab) using Plotly, and embedded here to deliver an interactive, data-driven web experience.

Data Source

The dataset used in this project was sourced from Kaggle’s Google Play Store Apps dataset . It includes detailed information on app ratings, reviews, installs, genres, size, and more. We performed extensive cleaning, preprocessing, and analysis in Python to extract meaningful patterns and visualize them interactively using Plotly.
